# docs/decisions/ — architecture decision records

Why things are the way they are, so nobody (human or AI) relitigates or accidentally undoes
them. One file per decision: `YYYY-MM-DD-<slug>.md`, from [TEMPLATE.md](TEMPLATE.md).

Write an ADR when a choice was made between real alternatives and the reasoning isn't obvious
from the code — stack picks, branch model changes, build-vs-buy, anything you'd otherwise
re-debate in six months. Infra-specific decisions live in `infra/decisions/` (same template).

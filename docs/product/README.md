# docs/product/ — roadmap & idea inbox

Not a team — the owner's product thinking, kept in files so the AI can read it and ideas
survive between sessions.

- [roadmap.md](roadmap.md) — now / next / later. The AI consults it to judge scope and
  priority; the owner reorders it in chat ("move X to now").
- [ideas.md](ideas.md) — append-only inbox. Owner drops one-liners any time; when an idea
  graduates, it becomes a ticket (and moves to roadmap or gets deleted).

Any team may **propose** here via chat or ticket; only the owner reorders the roadmap.
